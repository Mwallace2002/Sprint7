FRONTEND
- Instalar yarn
- Abrir el documento
- Cd frontend
- Yarn (descargar dependencias)
- Yarn run dev
- yarn add i18next
- yarn add react-i18next


BACKEND 
- Instalar express (npm install express) a la carpeta global
- Control c pa cerrar la server
- Para correrlo:
  - Cd server
  - ./src/index.js
